Estimating prior variance with asymptotic SER

When fitting the single effect regression (SER), we often want to take an empricial Bayes approach and estimate the prior variance of the effect. However, when we are using the asymptotic approximation, the quality of the approximation may vary for different choices of the prior variance. Here, we describe EB/Type-II maximum likelihood approach of maximizing the prior variance. We discuss the impact of the varying approximation quality.

Good where it counts?

We are interested in maximizing the evidence of the SER w.r.t the prior variance. Equivalently, we can think of maximizing the Bayes factor for the SER (that is, the ratio of the evidence of the single effect model vs the null model of no effect).

\[ BF_{SER}(\sigma_0^2) = \sum \pi_j BF_j(\sigma_0^2) \]

In practice we are using the asymptotic approximation. We approximation each univariate Bayes factor, \(BF_j\), with a Laplace approximation. In particular we take a quadratic approximation of the log-likelihood, and combine it with the normal prior. This gives us a method for approximating the MAP using only local information about the MLE (namely, the MLE \(\hat\beta\), the standard error \(s^2\), and the log-likelihood of the data under the MLE \(\log p ({\bf y} | \hat\beta)\)).

There are two things we should be concerned with. First, the quality of the posterior mean approximation. Second, the quality of the Bayes factors. In particular since we are maximizing over the prior variance, but we should not expect the quality of these approximations to be uniform for different settings of the prior variance.

  1. What settings of the prior variance lead to the best approximation quality? I expect that it will be for for large settings of the prior variance, where there is not much shrinkage the MAP is close to the MLE.
  2. How does the optimizing the prior variance impact the coverage of the GLM-SER? In particular, we can see the the BF for the SER is an average of the BF for the univariate regressions. If there a large number of variables with little evidence of association, these will draw down the optimal setting of \(\sigma^2_0\), and impose stronger shrinkage on the variables with strong evidence for an effect. On the other hand, if the univariate BF for the strongest effects are most sensitive to changes in \(\sigma^2_0\) we might see that the optimal choice of the BF is driven by a few variables. We’ve been burned by non-uniform approximation quality before (with the Jaakkola Jordan bound)– we want to be careful to make sure that estimating the prior variance does not introduce similar problems.

Optimal \(\sigma^2_0\) in the univariate case

Note we can equivalently optimize the ABF, rather than the Laplace ABF because these two approximations of the BF are just rescaled versions of one another.

\[ \hat \sigma^2_0 = \arg \max_{\sigma^2_0} N(\hat\beta | 0, s^2 + \sigma^2_0) \]

\(\nu = s^2 + \sigma^2_0\) is maximized by \(\nu = \hat\beta^2\), so for \(\hat\beta^2 > s^2\), we can take \(\hat \sigma^2_0 = \hat \beta^2 - s^2\). Otherwise, we run into the boundary and we should take \(\hat\sigma^2_0 = 0\).

# log_abf is a function of prior variance
log_abf <- function(betahat, s2){
  f <- function(prior_variance){
    dnorm(betahat, 0, sd = sqrt(s2 + prior_variance), log = T) -
      dnorm(betahat, 0, sd = sqrt(s2))
  }
  f2 <- function(x){}
  return(f)
}

Optimal \(\sigma^2_0\) for SER

$$ \[\begin{aligned} \hat \sigma^2_0 &= \arg\max_{\sigma^2_0} \sum_j \pi_j ABF_j(\sigma_0^2) \\ &= \arg\max_{\sigma^2_0} \sum_j \pi_j \exp\left\{-\frac{1}{2} \left(\log 2\pi(s^2_j + \sigma^2_0) + \frac{\hat\beta^2_j}{s^2_j + \sigma^2_0} \right)\right\} \\ \end{aligned}\]

$$

We can optimize directly, e.g. using optim. Alternatively we can optimize a variational lower bound.

\[ \begin{aligned} \log p(y) &\geq \mathbb E_q[\log p (y | b, \gamma) + \log p(b | \sigma^2_0) + \log p(\gamma)] + H(q(b, \gamma)) \\ &= \mathbb E_q[\log p (b | \sigma^2_0)] = \frac{1}{2} \log \tau - \frac{\tau}{2} \sum_j \alpha_j (\mu_j^2 + \sigma_j^2) \end{aligned} \]

Which has the update \(\frac{1}{\tau^*} = \sum_j \alpha_j (\mu_j^2 + \sigma_j^2)\). To fit the SER with the asymptotic approximation, we can compute the MLE, standard error, and log likelihood once for each of the \(p\) variables, then iteratively update \(q(b, \gamma; \sigma^2_0)\) and \(\sigma^2_0\) until convergence.

We show three examples– one where the optimal \(\sigma^2_0\) is 0 (because all the \(z\) scores are small), which exhibits slow convergence. The other \(\sigma^2_0 > 0\), and we see fast convergence even though the evidence for the single effect model is weak– we get very close to optimal in 3 iterations. In the final example the effect is strong, and we also converge on the optimal setting of \(\sigma^2_0\) quickly.

Code 
logsumexp <- function(x){
  a <- max(x)
  return(log(sum(exp(x - a))) + a)
}

compute_posterior_mean <- function(betahat, s2, prior_variance){
  tau0 <- 1 / prior_variance
  tau <- 1/s2
  
  tau_post <- tau + tau0
  mean_post <- (tau/tau_post) * betahat
  post <- list(mean = mean_post, tau=tau_post)
  return(post)
}

compute_log_bf_ser <- function(betahat, s2, prior_variance, pi){
  log_abf <- dnorm(betahat, 0, sd = sqrt(s2 + prior_variance), log = T) -
    dnorm(betahat, 0, sd = sqrt(s2), log = T) 
  logsumexp(log_abf + log(pi))
}

fit_prior_variance_em <- function(betahat, s2, prior_variance, pi, maxit=20){
  hist <- rep(0, maxit)
  log_bf <- rep(0, maxit)
  
  hist[1] <- prior_variance
  log_bf[1] <- compute_log_bf_ser(betahat, s2, prior_variance, pi)
  for(i in 2:maxit){
    # update alpha
    log_abf <- dnorm(betahat, 0, sd = sqrt(s2 + prior_variance), log = T) -
      dnorm(betahat, 0, sd = sqrt(s2), log = T)
    alpha <- exp(log_abf - logsumexp(log_abf))
    
    # e step
    post <- compute_posterior_mean(betahat, s2, prior_variance)
    b2 <- post$mean^2 + 1/post$tau
  
    # m step, weighted because of heteroskedactic observations
    prior_variance <- sum(alpha * b2)
    hist[i] <- prior_variance
    log_bf[i] <- compute_log_bf_ser(betahat, s2, prior_variance, pi)
  }
  
  return(list(prior_variance = prior_variance, log_bf = log_bf, hist=hist))
}

optimize_prior_variance <- function(betahat, s2, pi){
  f_opt <- function(prior_variance){
    compute_log_bf_ser(betahat, s2, prior_variance, pi)
  }
  opt <- optimise(f_opt, c(0, 20), maximum = T)
  return(opt)
}

plot_em <- function(p = 50, var = 0.25){
  prior_variance <- 1
  pi <- rep(1/p, p)
  betahat <- rnorm(p) * sqrt(var)
  s2 <- rbeta(p, 0.5, 0.5) + 0.1
  
  opt_res <- optimize_prior_variance(betahat, s2, pi)
  em_res <- fit_prior_variance_em(betahat, s2, 1, pi, maxit = 20)
  
  par(mfrow = c(1,2))
  plot(em_res$hist, ylim = range(em_res$hist, opt_res$maximum), xlab='Iter', ylab='prior variance')
  abline(h=opt_res$maximum, lty=3)
  
  plot(em_res$log_bf, ylim = range(em_res$log_bf, opt_res$objective), xlab='Iter', ylab='log BF')
  abline(h=opt_res$objective, lty=3)
}

Error comparison

Next we will look at the error in posterior mean and BF as a function of \(\sigma^2_0\). For the BF comparison we can compute the exact BF via quadrature. For the posterior mean we can compare to the MAP and the posterior mean of the optimal Gaussian approximation (not necessarily the same).

  1. ABF
  2. Laplace ABF
  3. Exact log likelihood

In the example shown, the exact log BF and the Laplace ABF are both fairly insensitive to \(\sigma^2_0\) across a range containing the maximum, and are both quiet close– and the approximation error is pretty uniform across reasonable choices of \(\sigma^2_0\). That is encouraging!

Code 
sigmoid <- function(x){1/(1 + exp(-x))}

simulate <- function(n, beta0=0, beta=1){
  x <- rnorm(n)
  logit <- beta0 + beta*x
  y <- rbinom(n, 1, sigmoid(logit))
  return(list(x=x, y=y, beta0=beta0, beta=beta))
}

compute_exact_log_bf <- function(x, y, 
                                 betahat, shat2, intercepthat,
                                 prior_variance=1){
  f <- function(b){
    logits <- x * b + intercepthat
    sum(dbinom(x=y, size=1, prob=sigmoid(logits), log = T)) + 
      dnorm(b, 0, sqrt(prior_variance), log = T)
  }
  max <- f(betahat)
  
  f2 <- function(b){purrr::map_dbl(b, ~exp(f(.x) - max))}
  
  optim_range <- c(-2*abs(betahat), 2 *abs(betahat))
  MAP <- optimize(f2, optim_range, maximum = T)$maximum
  
  lower <- MAP - sqrt(shat2) * 6
  upper <- MAP + sqrt(shat2) * 6
  quadres <- cubature::pcubature(f2, lower, upper, tol = 1e-10)
  log_bf <- (log(quadres$integral) + max) - sum(dbinom(y, 1, mean(y), log=T))
  return(log_bf)
}

compute_bfs <- function(x, y, prior_variance){
  fit <- glm(y ~ x, family='binomial')
  intercepthat <- fit$coefficients[1]
  betahat <- fit$coefficients[2]
  shat2 <- summary(fit)$coef[2,2]^2
  lr <- 0.5 * (fit$null.deviance - fit$deviance)
  
  # compute ABF
  log_abf <- dnorm(betahat, 0, sqrt(prior_variance + shat2), log = TRUE) -
    dnorm(betahat, 0, sqrt(shat2), log = TRUE)
  
  # compute liklihood ratio under asymptotic approximation
  alr_mle <- dnorm(betahat, betahat, sqrt(shat2), log=T) -
    dnorm(betahat, 0, sqrt(shat2), log=T)
  
  # corrected ABF
  log_labf <- log_abf - alr_mle + lr
  
  # compute exact log BF
  log_bf <- compute_exact_log_bf(x, y, 
                                 betahat, shat2, intercepthat,
                                 prior_variance)
  return(list(log_bf = log_bf, log_abf = log_abf, log_labf = log_labf))
}
